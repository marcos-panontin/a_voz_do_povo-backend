Bem-vindo (a) à API A Voz do Povo

Welcome to A Voz do Povo API

Essa API providencia as respostas de consulta ao Banco de Dados MySQL referentes à Aplicação FullStack A Voz do Povo

This API provides the answers to MySQL Database queries related to the FullStack Application A Voz do Povo

Endpoints

  • /answers - Mostrar todas as respostas dadas || Show all answers
  • /answers/:id - Mostrar a contabilidade de todas as respostas dadas para uma pergunta específica, substituindo o :id da URL pelo id da pergunta. || Show the count of all answers given to a specific question, replacing :id in the URL with the desired question id.
